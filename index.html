terminalCLIコマンド一覧

ターミナルから直接実行するコマンドです。

起動・基本
コマンド説明
claude インタラクティブモードで起動
claude "指示" 指示を1回実行して終了
claude -p "指示" 非インタラクティブ出力（パイプ・スクリプト向け）
claude --version バージョン確認
claude --help ヘルプ表示
会話の継続・再開
コマンド説明
claude -c 直前の会話を継続して起動
claude --continue 同上（長いオプション）
claude --resume 過去の会話を一覧から選んで再開
モデル・出力
コマンド説明
claude --model <model-id> 使用モデルを指定して起動
claude --output-format json 出力をJSON形式にする（-pと併用）
claude --output-format stream-json ストリーミングJSON形式で出力
claude --max-turns 5 自律実行の最大ターン数を指定
権限・ツール
コマンド説明
claude --allowedTools "Bash,Edit" 使用を許可するツールを指定
claude --dangerouslySkipPermissions 確認プロンプトをすべてスキップ（自動化用・要注意）
claude --add-dir /path アクセスを許可するディレクトリを追加

tune便利な設定

カスタマイズと動作制御のための設定ファイル・環境変数です。

description

CLAUDE.md

Claudeへの「お願い書き」。プロジェクトや作業スタイルの指示を記述する。
グローバル：~/.claude/CLAUDE.md
プロジェクト：.claude/CLAUDE.md

# プロジェクト指示の例 ## コーディング規約 - TypeScript strict mode を使う - コメントは日本語で書く ## 禁止事項 - console.log を本番コードに残さない
security

settings.json — パーミッション

ツール実行の許可・拒否を設定。確認プロンプトを減らせる。
場所：.claude/settings.json

{ "permissions": { "allow": [ "Bash(git *)", "Bash(npm run *)", "Bash(ls *)" ], "deny": [ "Bash(rm -rf *)" ] } }
smart_toy

利用可能なモデル

用途に応じてモデルを使い分ける。

  • star claude-opus-4-7最高性能
  • speed claude-sonnet-4-6バランス
  • flash_on claude-haiku-4-5-20251001高速・軽量
webhook

Hooks — 自動実行

ツール実行前後に自動でシェルコマンドを走らせる。settings.json で設定。

{ "hooks": { "PostToolUse": [{ "matcher": "Write|Edit", "hooks": [{ "type": "command", "command": "npm run lint --silent" }] }] } }
key

環境変数

APIキーや動作制御に使う主要な環境変数。

  • vpn_keyANTHROPIC_API_KEY — APIキー
  • cloudANTHROPIC_BASE_URL — エンドポイント変更
  • bug_reportCLAUDE_CODE_DEBUG — デバッグログ出力
  • terminalBASH_DEFAULT_TIMEOUT_MS — Bashタイムアウト
integration_instructions

MCP サーバー

外部ツールをClaudeに接続する拡張機構。settings.json に追加。

{ "mcpServers": { "my-server": { "command": "npx", "args": ["-y", "@my/mcp-server"], "env": { "API_KEY": "..." } } } }
